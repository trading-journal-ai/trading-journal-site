# Daily Recaps & Tags

This is the heart of the habit: a short written recap of the day, and a consistent set of
tags on each trade. Done daily, it turns scattered sessions into patterns you can actually
see.

## Writing a daily recap

Open a day and write a few sentences before you dig into individual trades:

- What was the **market read**, and did you trade with it or against it?
- How was your **execution** — patient, rushed, oversized?
- What's the **one lesson** you'd tell yourself tomorrow?

Keep it short. A recap you'll actually write every day beats a perfect one you write once a
month.

## Tagging a trade

On each trade, write a sentence in your own words, then tap the pills that fit. Think in
four buckets:

- **Quality** — one call on the trade overall (*Best setup*, *Good trade*, *Needs review*,
  *Rule break*, *Chased*).
- **Setup** — the pattern you were trading (*VWAP reclaim*, *HOD retest*, *Opening range*).
- **Process** — what you did (*Followed plan*, *Patient*, *Focused*, *Cut loss*,
  *Oversized*, *Moved stop*).
- **Emotion** — how you felt (*Calm*, *Confident*, *Impatient*, *FOMO*).

## Why consistency matters

The value isn't in any single tag — it's in using the **same vocabulary every session**.
Once you do, questions like these become answerable:

- How do my *FOMO* trades actually perform?
- When I tag *Followed plan*, is my win rate really higher?
- Which setup do I tag *Best setup* most, and is it my most profitable?

<Note title="Pick a small vocabulary and stick to it">
A dozen tags you use consistently are far more useful than fifty you use once. Start small;
add a tag only when you keep reaching for a concept that isn't there yet.
</Note>

## What's landing around this

Recaps and tagging work today, and two things are growing out of them:

- **Dictation** — speak the context behind a trade instead of typing it. A first version
  is live in ticker review, transcribed by a local speech model so audio never leaves your
  machine, and it's expanding to more surfaces.
- **Coach review** — with your own model API key configured, the [coach](/docs/guide/ai-coach)
  can already draft a review of your day from these same notes and tags, and that
  integration is deepening on the [roadmap](/docs/roadmap).
